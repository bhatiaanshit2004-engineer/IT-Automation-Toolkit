# IT-Automation-Toolkit
PowerShell-based IT automation toolkit for Windows administration, system monitoring, security auditing, and network troubleshooting.
A collection of PowerShell scripts for Windows administration and network troubleshooting.

## Features

### Windows Scripts

- System Information Collection
- Event Log Analysis
- Performance Monitoring
- Service Audit
- Security Compliance Check
- User & Group Audit
- Windows Update Status

### Network Scripts

- Ping Testing
- DNS Resolution Testing
- Port Availability Testing
- Traceroute Diagnostics
- Network Monitoring
- Internet Availability Detection
- Network Configuration Collection
- Automated Report Generation

## Project Structure

```
IT-Automation-Toolkit/
```

## Requirements

- Windows 10/11
- PowerShell 5.1+
- Internet connection

## Usage

Run any script:

```powershell
.\windows-scripts\system-info.ps1
```

Reports are stored in:

```text
test-results/windows/
```
