# Installation Guide

## Prerequisites

- Windows 10 or Windows 11
- PowerShell 5.1 or later
- Git
- Visual Studio Code

## Clone Repository

```bash
git clone https://github.com/bhatiaanshit2004-engineer/IT-Automation-Toolkit.git
```

## Open Project

```bash
cd IT-Automation-Toolkit
code .
```

## Enable Script Execution

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```
